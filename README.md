# React-Login

- a secure login with the use of JWT 
- an account creation page that checks many validation fields
- a responsive web application 

```bash
git clone 'RemoteName'
```
## start react
```bash
cd ./website
npm install
sudo npm start
```
## start api
```bash
cd ./API
npm install
sudo node app.js
```

## update
```bash
sudo npm outdated
```
